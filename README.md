# C Tutorial Programs

This repository is a beginner-friendly set of standalone C programs. Each file focuses on one concept and has its own `main()` function, so compile and run one lesson at a time.

## Requirements

- GCC or another C compiler
- A terminal or an editor such as VS Code
- Optional: `make` for building all examples into the `build/` folder

## Quick Start

Compile one lesson:

```sh
gcc -std=c11 -Wall -Wextra -pedantic 1_Introduction.c -o build/1_Introduction
```

Run it:

```sh
./build/1_Introduction
```

On Windows, the output file usually ends with `.exe`:

```sh
gcc -std=c11 -Wall -Wextra -pedantic 1_Introduction.c -o build/1_Introduction.exe
build\1_Introduction.exe
```

Build every lesson with Make:

```sh
make
```

On Windows/MSYS2, use:

```sh
make EXEEXT=.exe
```
